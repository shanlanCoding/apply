# 基于JavaWeb架构的在线报名系统

### 后台：
- Spring Boot： 消除复杂的xml配置文件
    - Spring framework： 管理Bean
    - Spring MVC： Web层页面跳转、管理
- Mybatis 持久层
- Spring Security 安全框架
- Thymeleaf 页面模板


### 前台：
- Bootstrap 页面搭建
- jQuery 页面控制


# 开发环境：
- JDK8 程序语言
- IDEA2019.1 开发编辑器 | Eclipse
- MySQL 数据库
- Maven 依赖管理
- Git 版本控制
---------

### 注意：clone本项目后，请先建立wcgxy数据库，然后将“DataBase.sql”文件恢复到数据库中。
